# Mariner TUI Requirements Summary

Mariner TUI is a terminal applicaiton used for displaying NOAA weather and tide information for a given port.

* US based Only
* Displays current weather and next 3 days
* It must include NOAA wind predictions
* It must display wave heights
* It must display any alerts from NOAA for that port
* Displays tides for next 3 days
* Display should be in color and am open to using UTF emoji for varios weather conditions
* Each data section should be in a pane view
* Ability to search ports by postal code or city/state
